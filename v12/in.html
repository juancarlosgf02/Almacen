Ayuda

  • done¿Que hacer si somos nuevos en este sistema?

    Debemos seguir los siguientes pasos para crear un nuevo usuario:
    1. Debemos dirigirnos a la opcion REGISTRATE AQUI.
    2. Llenar los siguientes campos con nuestra informacion.
    3. Procesar los datos para el creado del usuario.
    4. Aceptacion del Usuario.
    5. Finalizar.
    Usted ha creado satisfactoriamente su nueva cuenta de usuario.

  • done¿NO se puede ingresar a la cuenta del usuario?

    1. Debemos verificar si estamos escribiendo bien la contraseña o el usuario.
    2. Verificar las mayusculas, giones, punto u otro caracter.
    3. Verificar si la base de datos sigue activa.

  • done¿Problemas al iniciar?

    1. Verificar si la conexiones estan activas.
    2. Verificar si el sitio se encuentra accesible.
    3. Verificar si el usuario esta activado.
